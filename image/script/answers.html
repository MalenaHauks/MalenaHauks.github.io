1. Rastergrafik: Bilder som byggs upp av enskilda pixlar. Deras främsta nackdel att de blir gryniga(pixelerade) om de förstoras eller förminskas. Vektorgrafik: Bilder som byggs upp genom banor (som definieras matematisk). Används för logotyper, typsnitt, ritningar eftersom de kan förstoras och förminskas utan att dess kvalité försämras. 2. JPEG(Joint Photographic Experts Group), komprimerad, stödjer ej alpha (genomskinlighet) BMP (Windows Bitmap) (*.bmp) ej komprimerad som i sin tur ger stora filer GIF(Graphic Interchange Format)(*.gif), bara 256 färger, stödjer animering PNG(Portable Network Graphics)(*.png), manga färger, bra komprimering, stödjer alpha. TIFF(Tagged Image File Format) (*tif) för foto ock tryck SVG (Scalable vector graphic)(.svg eller inbäddad i html) för vektorgrafik
